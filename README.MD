# Offline Vocabulary Flashcard App (11-Language Support)

A lightweight browser-based vocabulary memorization app designed for serious learners.

This project was originally created to memorize English vocabulary while preparing for the Korean civil service exam, where reading comprehension is the most important skill.  
Because memorizing thousands of words was difficult, this tool was built to make vocabulary learning faster, easier, and more systematic.

Now it supports **11 languages**, so it can also help vocabulary learning for many other languages.

---

## Features

- Flashcard-style vocabulary learning
- Dark mode support
- Spaced repetition style review
- Word status tracking (learning / wrong / mastered)
- Fully offline (no installation required)
- Lightweight & fast (HTML + JavaScript)
- Multi-language support (11 languages)

---

## How to Use

1. Download or clone this repository
2. Open `index.html` in your browser
3. Load your vocabulary file
4. Start studying

No installation required. Works fully offline.

---

## Why This Project Exists

While preparing for the Korean civil service exam, I struggled to memorize a large amount of vocabulary.  
Since reading comprehension is critical in the exam, vocabulary memorization became essential.

To solve this problem, I built this app as a personal study system.

If this project helps other learners studying English or any other language, I will be very happy.

# 한국어 설명

## 오프라인 단어 암기 앱 (11개국 언어 지원)

이 프로젝트는 공무원 시험 준비 중 영어 독해 비중이 높아 어휘 암기가 필요했지만 어려워서  
효율적인 단어 암기를 위해 직접 만든 개인 학습용 단어장입니다.

현재 **11개국 다국어 지원**으로 영어뿐 아니라 다양한 언어의 어휘 학습에도 활용할 수 있습니다.

---

## 주요 기능

- 깜박이 방식 단어 학습
- 다크모드 지원
- 복습 중심 학습 구조
- 학습 / 오답 / 암기 상태 관리
- 완전 오프라인 실행
- HTML + JavaScript 기반 초경량
- 11개국 언어 지원

---

## 사용 방법

1. 저장소 다운로드 또는 클론
2. `index.html` 실행
3. 단어 파일 불러오기
4. 학습 시작

설치 없이 브라우저에서 바로 실행됩니다.

---

## 만든 이유

공무원 시험은 영어 독해 비중이 높아 많은 어휘 암기가 필요했습니다.  
하지만 수천 개 단어를 외우는 과정이 매우 어려워, 더 쉽고 체계적인 암기 도구가 필요했습니다.

그래서 개인 학습용으로 이 앱을 만들었습니다.

이 프로젝트가 영어 또는 다른 언어를 공부하는 분들에게 도움이 된다면 기쁠 것입니다.

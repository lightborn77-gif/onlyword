# OnlyWord — 깜박이 & 퀴즈 단어암기 (PC / Mobile)

**브라우저만 있으면 바로 실행되는 단어 암기 훈련 앱**입니다.  
단어장을 계속 펼쳐서 ‘의지로’ 외우는 방식이 힘들어서, **그냥 실행만 하면 반복 노출(깜박이) + 반복 테스트(퀴즈)**로 암기가 굴러가도록 만들었습니다.

저는 공무원 영어 시험을 준비하는 입장에서 이 앱으로:
- **12일**: Word Master 2000 정답률 **92%**
- **18일**: 정답률 **99%**
- 하루 **약 40분** (깜박이 1회 + 퀴즈 1회), **누적복습(따로 모아서 복습) 없이**

성과를 만들었습니다. 수능/토익/토플/공무원 등 시험 준비, 또는 영어 원서 독서를 위해 어휘를 빠르게 올리고 싶은 분들에게도 도움이 될 거라 생각합니다.

또한 현재 **11개 언어(학습 언어 선택)**를 지원해, 영어뿐 아니라 다른 외국어 단어 암기에도 활용할 수 있습니다.

---

## 1) 실행 방법 (설치 없음)

### PC 버전
1. `pc/index.html` 을 더블 클릭(또는 브라우저로 열기)
2. `📂 파일 로드` 버튼으로 **단어장 txt**를 선택

### Mobile 버전
1. `mobile/index.html` 을 모바일 브라우저로 열기
   - iOS는 “홈 화면에 추가”하면 앱처럼 사용 가능
2. 화면의 **파일 선택**으로 단어장 txt 로드

> 로컬 파일로 열어도 동작합니다(파일 읽기: FileReader, 저장: localStorage).

## 2) 단어장 파일 형식 (가장 중요)

이 앱은 **번호(1,2,3...)를 기준으로 ‘영어/뜻’을 한 세트로 묶습니다.**  
아래 형식 중 아무거나 사용하면 됩니다.

### ✅ 형식 A (2줄 세트)
```
1. apple
1. 사과
2. arrive
2. 도착하다
```

### ✅ 형식 B (같은 번호로 여러 뜻)
```
1. run
1. 달리다
1. (기계가) 작동하다
2. break
2. 부수다
```
- 같은 번호로 뜻이 여러 줄이면 자동으로 묶어서 저장됩니다.

### ✅ 형식 C (점(.) 대신 공백/하이픈도 허용)
```
1 apple
1 사과
2 - arrive
2 - 도착하다
```

### 🔎 판별 규칙 (로직)
- **각 줄의 맨 앞**에서 `번호`를 읽습니다. (예: `1`, `23`)
- 같은 번호끼리 묶어서 한 항목을 만듭니다.
- 묶인 텍스트가
  - **영문만 포함**(알파벳 있고 한글 없음)이면 `학습단어(eng)`
  - **한글이 포함**되면 `해석뜻(kor)`
- 최종적으로 `eng`와 `kor` 둘 다 있는 항목만 유효 단어로 로드됩니다.

> **( ) 괄호 내용**, 일부 아이콘(✅🟢❌)은 로드 시 제거될 수 있습니다(PC 버전 기준).

## 3) 주요 기능 요약

### 공통 (PC/Mobile)
- **깜박이(자동 반복 노출)**: 설정한 속도로 단어/뜻을 반복 표시
- **퀴즈 모드**: 영→한 / 한→영 / 혼합, 섞기, 정답 표시, 오답 지연
- **SRS 복습**: 간격 기반 반복(학습 기록을 사용)
- **TTS 발음**: 음성 선택, 속도/피치 조절, 자동발음 옵션
- **필터/범위 학습**: 시작~끝 번호 범위 지정 + 조건 필터
- **학습 데이터 저장**: 브라우저 localStorage에 자동 저장(파일명 기준)

### PC 전용(주요)
- 다크모드, 전체화면/학습창 전용 전체화면
- 학습 통계 패널(오늘/주간, 안정권 비율, 정답률, 연속일수, 최근 7일, TOP 오답)
- 모아보기(리스트 뷰) + 자동 재생
- 세이브/로드(JSON 내보내기/가져오기)

### Mobile 전용(주요)
- 모바일 UI(칩형 설정/팝업)
- 학습창 풀스크린(플로팅 버튼)
- 퀴즈 설정 팝업, SRS 설정 팝업, 통계 모달

---

## 4) 데이터 저장/복원

- 기본적으로 **파일명 기준**으로 `mem_<파일명>` 키로 localStorage에 저장됩니다.
- PC 버전은 추가로:
  - **세이브**: 현재 상태를 JSON 파일로 내보내기
  - **로드**: JSON 파일을 가져와서 복원

---

## 5) 도구: 단어 변환기 (logicmaker)

`tools/logicmaker.html` 은 **텍스트 단어장을 앱 입력 형식으로 정리/변환**하는 도구입니다.
- 불필요한 기호/주석 제거
- 번호 붙이기/형식 맞추기
- 범위 지정 변환 등

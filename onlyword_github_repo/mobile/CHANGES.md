# 회화 모드 제거 및 TTS 설정 이동

## 📋 수정 내용

### 1. 회화(Dialog) 모드 제거
- **모드 선택 버튼 제거**
  - 메인 화면의 "회화" 모드 버튼 삭제
  - 풀스크린 퀵 모드바의 "회화" 버튼 삭제

- **회화 컨트롤 바 제거**
  - 회화 설정 버튼 제거
  - 회화 시작 버튼 제거

- **회화 설정 팝업 제거**
  - 역할 설정 (A/B) 제거
  - 대기 시간 설정 제거
  - 쉐도잉 모드 설정 제거

- **JavaScript 파일 비활성화**
  - `dialog_state.js` - 주석 처리
  - `dialog_flow.js` - 주석 처리
  - `dialog_controls.js` - 주석 처리

### 2. TTS 설정 메뉴로 이동
- **메뉴 팝업에 TTS 설정 버튼 추가**
  - 위치: 설정 메뉴(⚙️) 상단
  - 버튼: "🎤 음성 설정 (TTS)"
  - 다국어 음성 지원을 위해 전역 설정으로 이동

## 💡 사용 방법

### TTS 설정 접근
1. 헤더의 설정 버튼(⚙️) 클릭
2. "🎤 음성 설정 (TTS)" 버튼 클릭
3. 학습 언어 선택 (영어, 스페인어, 프랑스어 등)
4. 음성 선택 및 속도/피치 조정

### 기존 기능 유지
- 깜박이 모드 ✅
- 퀴즈 모드 ✅
- SRS 복습 모드 ✅
- TTS 음성 재생 ✅
- 다국어 지원 ✅

## 🔍 주의사항

1. **회화 스크립트 파일**: 회화 관련 JS 파일들은 삭제하지 않고 로딩만 비활성화했습니다. 필요시 재활성화 가능합니다.

2. **하위 호환성**: 기존 데이터나 설정은 그대로 유지됩니다.

3. **TTS 기능**: TTS 음성 재생 기능은 모든 모드에서 정상 작동합니다.

# Mobile 사용법

## 1) 실행
- `mobile/index.html`을 모바일 브라우저로 열기
- iOS: 공유 → "홈 화면에 추가" 권장

## 2) 파일 로드
- 첫 화면의 `파일 선택` 버튼으로 txt 로드

## 3) 모드
- 깜박이 / 퀴즈 / SRS 복습을 상단 모드 버튼으로 전환
- 각 모드별로 상단 컨트롤 바(칩)를 눌러 범위/속도/크기/설정 조정

## 4) 하단 버튼
- ◀/▶: 이전/다음
- ⭐: 외움(안정권) 토글
- ▶: 자동 재생
- ★: 별표 토글

## 5) 저장
- 자동 저장: localStorage(파일명 기준)

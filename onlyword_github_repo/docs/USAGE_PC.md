# PC 사용법

## 1) 실행
- `pc/index.html` 열기

## 2) 파일 로드
- `📂 파일 로드 (자동감지)` → 단어장 txt 선택
- 로드가 완료되면 상단 상태 메시지에 로드된 개수가 표시됩니다.

## 3) 학습 모드
### 깜박이
- `시작/계속 (Space)`로 자동 재생
- `외움 (Ent)`로 안정권(암기) 표시
- 속도/글자 크기는 설정에서 조절

### 퀴즈
- 영→한 / 한→영 / 혼합 선택
- 문항수, 오답대기, 섞기, 정답표시 설정
- 별표 복습 포함(최근 N일) 옵션

### SRS
- 새 단어 포함 / 어려운 단어 우선
- 간격 배율, 실패 시 감소 방식 설정

## 4) 저장/복원
- 기본 자동 저장: localStorage(파일명 기준)
- 추가로 `세이브/로드`(JSON)로 다른 PC로 옮기기 가능
